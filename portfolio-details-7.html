Costume Expo Module

Project information

  • Project Name: Costume Expo Module
  • Project date: july 2023
  • Category: Mobile Simulation Module
  • Tools: Unity 3D, C #, Sketchfab, Blender, Ready Player Me,Mixamo
  • Project URL: https://youtube.com/shorts/6gINN5lWwJ4?feature=share

The Costume Expo Module stands as a testament to my innovation in the realm of Android applications, delivering a groundbreaking demo that redefines real-time costume changes. At the heart of this project lies the strategic use of models from Ready Player Me, seamlessly imported and meticulously crafted through a streamlined workflow from Blender to Unity. This venture not only showcases the sheer creativity and technical finesse behind the scenes but also provides users with an unparalleled experience. Navigating through an extensive array of virtual costumes, the application offers an immersive journey, transcending the boundaries of traditional mobile experiences. By pushing the boundaries of technological integration, the Costume Expo Module not only entertains but serves as a pioneering example of the seamless synergy between Ready Player Me's models and the Android platform, setting a new standard for visually compelling and technologically advanced applications.
